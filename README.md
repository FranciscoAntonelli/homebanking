# Homebanking - Proyecto académico grupal

Sistema de gestión tipo homebanking desarrollado como trabajo práctico en la UTN.

## Tecnologías
- Java + Spring Boot (backend, API REST)
- MySQL (base de datos relacional)
- JavaScript (interacción frontend)
- HTML y CSS (estructura y estilos)

## Funcionalidades
- Consulta de información del cliente mediante DNI
- Visualización de cuentas asociadas
- Depósitos y retiros de efectivo
- Transferencias entre cuentas
- Historial de movimientos
